# Product Overview

## 产品定位

JobFit 是一套面向求职面试准备场景的智能面试教练 Agent。它以目标岗位和个人经历为核心，帮助用户完成岗位理解、能力诊断、模拟面试和复盘提升。

产品不是简单的 AI 问答工具，而是将真实求职准备流程拆解为连续的 Agent 工作流，让 AI 在明确业务目标和规则约束下完成分析、决策和训练任务。

## 用户问题

求职者在准备面试时通常会遇到以下问题：

- 不清楚目标岗位真正考察什么能力。
- 不知道自己的经历和岗位要求差在哪里。
- 网上题目零散，缺少岗位和能力维度的组织。
- 回答后无法判断是否具体、是否贴合岗位。
- 练习结束后缺少可执行的提升路径。

## 产品方案

JobFit 将求职准备拆成五个核心阶段：

1. 职位分析：生成目标岗位能力模型。
2. 背景分析：生成个人能力画像。
3. 能力差距：识别岗位要求和个人现状之间的差距。
4. 模拟面试：基于能力差距组织针对性练习题。
5. 复盘提升：基于答题记录生成数据分析和下一步练习方向。

## 核心价值

- 让用户知道目标岗位需要什么。
- 让用户知道自己缺什么。
- 让练习题更贴近真实面试场景。
- 让每次回答都能沉淀为下一轮提升依据。
- 让 AI 从单次内容生成升级为持续陪练的 Agent 工作流。

# Agent Workflow

JobFit 的核心是一个围绕求职面试准备的连续 Agent 工作流。

## 工作流总览

```text
岗位输入
  -> 岗位能力模型
  -> 用户背景输入
  -> 个人能力画像
  -> 能力差距分析
  -> 面试题组织
  -> 用户作答
  -> AI 点评
  -> 复盘分析
  -> 下一轮练习方向
```

## 阶段说明

### 1. 岗位理解

Agent 根据岗位名称、JD、行业和补充信息，生成岗位族、职责、核心能力、关键技能和面试关注点。

### 2. 候选人分析

Agent 根据用户经历、项目经验和补充信息，识别已有能力、优势信号、薄弱点和可迁移能力。

### 3. 能力差距诊断

Agent 将岗位能力模型和个人能力画像进行匹配，输出匹配状态、证据强度、差距原因、面试风险和练习重点。

### 4. 模拟面试

系统根据能力差距组织练习题，并为每道题展示考察点、答题框架和参考回答示例。

### 5. 回答点评

Agent 判断回答是否达到可点评标准。有效回答会进入维度评分、主要问题、优化建议和复盘数据。

### 6. 复盘提升

系统基于本轮答题结果生成综合表现、失分原因、能力短板、答题质量分布和下一批练习方向。

## 设计原则

- 先理解岗位，再训练用户。
- 先识别差距，再组织题目。
- 先判断回答是否有效，再进行点评。
- 复盘结果必须服务下一轮练习。
- AI 输出应当有规则、有来源、有结构，而不是自由发挥。
